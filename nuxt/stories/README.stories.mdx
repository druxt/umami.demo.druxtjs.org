# Druxt Umami Storybook

This is the Storybook for the Druxt Umami Parity project:
https://demo.druxtjs.org

The Druxt Umami Parity project is an example of how to build a fully decoupled
Drupal site with a Nuxt.js frontend using [Druxt](https://druxtjs.org).

## Stack

This site is built with the following techology stack:
- **[BootstrapVue](https://bootstrap-vue.org)**: Bootstrap UI framework for Vue
- **[Drupal](https://drupal.org)**: PHP Content management system
- **[Druxt](https://druxtjs.org)**: Fully decoupled Drupal framework for Nuxt.js
- **[Nuxt.js](https://nuxtjs.org)**: Vue Frontend framework

## Druxt Storybook

> Druxt + @nuxtjs/storybook = autogenerated stories

Druxt is connected to the Drupal backend via the JSON:API, allowing the Druxt
modules to autogenerate stories based on Drupals configuration.

For more information see https://druxtjs.org/guide/storybook

## Opensource

The source code for the Druxt Umami demo and storybook can be found @ https://github.com/druxt/demo.druxtjs.org
